# Bibliography for ROS 2 Lessons

This directory contains APA-formatted citations for the ROS 2 educational lessons.

## How to Add Citations

When adding content to the ROS 2 lessons, ensure all technical claims are source-traceable using proper APA citation format.

### Examples of APA Format:

- Book: Author, A. A. (Year). *Title of work*. Publisher.
- Journal Article: Author, A. A., & Author, B. B. (Year). Title of article. *Title of Periodical*, volume(issue), pages. https://doi.org/xx.xxx/yyyy
- Web Page: Author, A. A. (Year, Month Date). Title of online content. *Website Name*. URL

## Citation Tracking

Each lesson should have its own references section at the end with properly formatted citations.

## Tools

Consider using citation management tools like Zotero or Mendeley to maintain consistent formatting.
